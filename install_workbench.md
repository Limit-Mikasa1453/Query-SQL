## INSTALL MYSQL WORKBENCH for editor mysql
> Jalankan terminal linux
```ruby
$ sudo apt update
```
```ruby
$ sudo apt install mysql-workbench
```
menampilkan user interface/aplikasi workbench
```ruby
$ mysql-workbench 
```
